ai_cpp - ии разработонной мною, для работы ии необходимы умение компилировать файлы с помощью мейкфайл.
так же умение подключать свои ключи через консоль для работы.
инфа о файлах cpp
- main.cpp — точка входа, цикл ввода/вывода, работа с MemoryDB.
- ai.h / ai.cpp — логика обработки запроса: поиск в памяти, похожие ответы, интернет, обучение.
- memory.h / memory.cpp — класс MemoryDB на SQLite: методы save, search.
- web.h / web.cpp — модуль для интернет‑поиска через libcurl + API (или HTML).
- format.h / format.cpp — функции форматирования результатов и ошибок.
- utils.h / utils.cpp — вспомогательные утилиты (например, normalize).
Логика работы 
- Пользователь вводит запрос.
- processInput нормализует строку.
- Проверка в памяти (MemoryDB::search).
- Если нет — поиск похожих ключей.
- Если нет — запрос в интернет (fetchFromWeb).
- Если интернет не дал результата — ручное обучение (пользователь вводит ответ).
Инструменты необходимы для работы!
- SQLite3 — хранение памяти.
- libcurl — HTTP‑запросы.
- nlohmann/json — парсинг JSON (если используем API).
