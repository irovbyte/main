# ai_cpp

**ai_cpp** — это чат‑бот на C++, разработанный мной.  
Он умеет хранить знания в базе SQLite, искать похожие ответы и при необходимости обращаться к интернет‑поиску через Google Custom Search API.

---

## Возможности
- Запоминание новых ответов и ключевых слов.
- Поиск по памяти (SQLite).
- Поиск похожих ключей.
- Интернет‑поиск через API (libcurl + JSON).
- Ручное обучение, если ответа нет.

---

## Требования
- Установленные библиотеки:
  - `libsqlite3-dev`
  - `libcurl4-openssl-dev`
- Компиляция через **Makefile**:
  ```bash
  make clean
  make
  ./bot
  ```
- Каталог `data/` для хранения базы (`data/memory.db`).

---

## API‑ключи
Для работы интернет‑поиска необходимо подключить свои ключи от **Google Custom Search API**:

1. Получите ключ API и идентификатор поисковой системы (CX) в [Google Cloud Console](https://console.cloud.google.com/).
2. В файле `src/web.cpp` замените строки:
   ```cpp
   std::string apiKey = "YOUR_API_KEY";
   std::string cx     = "YOUR_CX";
   ```
   на свои значения.
3. Можно также передавать ключи через переменные окружения:
   ```bash
   export GOOGLE_API_KEY="ваш_ключ"
   export GOOGLE_CX="ваш_cx"
   ```
   и в коде использовать `getenv("GOOGLE_API_KEY")`.

## Структура проекта
```
ai_cpp/
 ├── Makefile
 ├── include/
 │    ├── ai.h        # Логика обработки ввода
 │    ├── memory.h    # Класс MemoryDB (SQLite)
 │    ├── web.h       # Интернет-поиск
 │    ├── format.h    # Форматирование результатов
 │    └── utils.h     # Вспомогательные функции
 ├── src/
 │     ├── main.cpp    # Точка входа
 │     ├── ai.cpp      # Логика ИИ
 │     ├── memory.cpp  # Реализация MemoryDB
 │     ├── web.cpp     # Интернет-поиск через API
 │     ├── format.cpp  # Форматирование
 │     └── utils.cpp   # Утилиты
 ├── data/
 │    └── memory.db   # база данных (создаётся автоматически)
 └── build/
      └── bot         # итоговый бинарник

```
